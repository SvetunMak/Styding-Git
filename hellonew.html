1. С помощью Git программисты и разработчики ориентируются в коде и отслеживают изменения/
  2. Git помогает вернуть файлы в исходное состояние и видеть изменения, внесенны в определенный период
  3. Разработчик выполняет разные команды (например, commit, push), а все изменения синхронизируются с центральным репозиторием.
  4. Git - это система контроля версий, а GitHub - онлайн-сервис, по сути социальная сеть.
